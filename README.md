# Shooting game

랜덤으로 날아오는 물체를 피하면서 총알을 쏴서 없애는 게임

날아오는 물체와 플레이어가 충돌했을 때 또는 HP가 0으로 됐을 때 게임이 종료된다.
물체가 화면 끝까지 와서 충돌하면 플레이어의 HP는 떨어진다.

실행환경 : Unity
